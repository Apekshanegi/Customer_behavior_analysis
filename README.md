Data Analytics Portfolio Project
📌 Overview

This project demonstrates the complete data analytics workflow, from data loading and preprocessing to business insights and dashboard creation. The analysis was performed using Python, SQL, and Power BI to transform raw data into meaningful insights for decision-making.

📂 Dataset
Dataset: Customer Shopping Behavior Dataset (replace with your dataset name if different)
Format: CSV
Description: Contains customer demographics, purchasing behavior, product details, and transaction information used for analysis and visualization.

🔄 Project Workflow
1. Data Loading
Imported the dataset into Python using Pandas.
Verified data types and dataset structure.
2. Exploratory Data Analysis (EDA)
Examined data distribution and summary statistics.
Identified trends, patterns, and relationships.
Created visualizations for better understanding.
3. Data Cleaning
Removed duplicate records.
Handled missing values.
Corrected inconsistent data.
Converted data into appropriate formats.
4. SQL Analysis
Imported the cleaned dataset into MySQL / SQL Server.
Wrote SQL queries to:
Filter and sort records
Aggregate data using GROUP BY
Perform joins
Calculate business metrics
Generate analytical insights
5. Power BI Dashboard

Created an interactive dashboard containing:

KPI Cards
Sales Analysis
Customer Insights
Product Performance
Category Analysis
Filters and Slicers
6. Report & Presentation
Prepared a project report summarizing the analysis.
Created a professional presentation using Gamma highlighting objectives, methodology, findings, and recommendations.
